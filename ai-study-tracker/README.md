# 🚀 AI Study Tracker

A modern productivity-focused study tracking web application built with Next.js and Tailwind CSS.

---

## 🧠 Overview

AI Study Tracker helps users set daily study goals, track their progress, and receive smart feedback based on recent performance.

The application includes a mock AI system that analyzes user activity over the last 3 days and provides productivity advice.

---

## ✨ Features

- ➕ Add study goals
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Persistent data with localStorage
- 📊 Progress tracking (completion rate)
- 🤖 AI-based (mock) productivity suggestions
- 🎨 Clean and modern UI (Tailwind CSS)

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **State Management:** React Hooks
- **Persistence:** localStorage
- **Deployment:** Railway

---

## 🤖 AI Logic (Mock)

The app includes a rule-based AI system:

- If most tasks are incomplete → suggests reducing workload  
- If most tasks are completed → suggests increasing difficulty  
- If mixed → suggests improving consistency  

This system is designed to be easily replaceable with a real AI API (e.g., OpenAI).
