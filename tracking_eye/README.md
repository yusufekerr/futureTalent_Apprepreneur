# TrackingEye

TrackingEye, yatırıma yeni başlayan kullanıcıların portfoylerini mobilde kolayca takip etmesini hedefleyen bir MVP urunudur.

## Proje Ozeti

Bu klasor, TrackingEye urunu icin urun gereksinimlerini (PRD) icerir.

- Urun: TrackingEye
- Surum: 1.0 (MVP)
- Platform: iOS / Android (cross-platform)
- Durum: Draft

## Problem

Yeni baslayan yatirimcilar yatirimlarini duzenli takip etmekte zorlanir. TrackingEye, tum varliklari tek yerde gostererek toplam deger, kar/zarar ve dagilim takibini kolaylastirir.

## MVP Hedefleri

1. Portfoye varlik ekleme
2. Toplam portfoy degerini gorme
3. Kar/zarar bilgisini otomatik hesaplama
4. Varlik dagilimini gorsellestirme
5. Kritik metrikleri tek dashboard ekraninda sunma

## MVP Kapsami (In Scope)

- Login/Register
- Varlik ekleme (ad, tur, adet, alis fiyati, guncel fiyat)
- Dashboard uzerinde toplam deger ve kar/zarar
- Portfoy detay ekrani
- Varlik dagilim grafigi

## MVP Disi (Out of Scope)

- Gelismis teknik analiz
- Sosyal yatirim ozellikleri
- Broker entegrasyonu ve otomatik al/sat
- Gelismis bildirim/oneriler
- Web surumu
